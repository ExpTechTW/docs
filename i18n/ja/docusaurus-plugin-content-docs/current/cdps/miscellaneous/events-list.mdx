# イベント一覧

利用可能なイベントを見る

## 目錄

- onServerStartEvent
- onServerCloseEvent
- onCommandEvent

## onServerStartEvent

#### 戻り値パラメータ`pid`

サーバー起動時

## onServerCloseEvent

#### パラメータ`reason（閉鎖の理由）`を返す

當伺服器被關閉
