# 命令
- CDPSは、対応する操作を実行するためのシンプルなCLIツールを提供する。
:::note
- 以下のコマンドはソースコードの下で、`cdps`を`python main.py`に置き換える必要があります
:::

## CDPSのヘルプメッセージを見る
- `-h` `--help`
```bash
cdps -h
```
:::note
- 子コマンドのヘルプメッセージを見ることができます。
- 例：`cdps pack -h`
:::

## CDPSバージョンを見る
- `-v` `--version`
```bash
cdps -v
```

## CDPSの初期化
```bash
cdps init
```

## CDPSの起動
- `パラメータなし` `start`
```bash
cdps start
```

## デフォルトのコンフィギュレーション（config.yml）を再生成する。
```bash
cdps gendefault
```

## パッケージ
```bash
cdps pack --name {フォルダ名}
```

## ホットリロード
```bash
cdps plugin reload {フォルダ名}
```
