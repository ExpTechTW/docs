## Manual Installation

Bundled files are included in the source repository.

1. Clone the repository and place it inside of your project.

   ```bash
   git clone https://github.com/ExpTechTW/api-wrapper.git
   ```

2. Import files from `dist` folder.

   `dist` folder contains three files and one folder:

   - [`/types`](https://github.com/ExpTechTW/api-wrapper/blob/main/dist/types) - All type definitions for TypeScript Intellisense
   - [`index.js`](https://github.com/ExpTechTW/api-wrapper/blob/main/dist/index.js) - CommonJS module file
   - [`index.min.js`](https://github.com/ExpTechTW/api-wrapper/blob/main/dist/index.min.js) - Minified CommonJS module file
   - [`index.mjs`](https://github.com/ExpTechTW/api-wrapper/blob/main/dist/index.mjs) - ESModule module file
