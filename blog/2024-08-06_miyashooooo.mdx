## 🌟 新增

- 使用者設定檔及初次啟動時自動新增預設設定檔
- 地圖所在地位置
- 左上方所在地即時測站資訊
- Windows 開機自行啟動自訂選項
- 地震預警來源自訂選項
- 語音播報自訂選項
- 地圖縮放自訂選項
- 顯示斷層自訂選項
- 登入狀態保存
- 重置設定按鈕
- 依照視窗自動調整地圖縮放
- 重播功能
- 地震詳細資訊顯示/隱藏同心圓按鈕
- 地震詳細資訊顯示各地震度及震央圖標
- 忘記密碼按鈕
- 購買 VIP 按鈕

## 🔌 最佳化

- 調用 API 請求
- 精簡部分程式碼

## 🐞 修正

- 預警來源顯示錯誤
- 地震結束後即時震度列表未隱藏
- 地震結束後沒有隱藏地圖上的各地震度
- 程式崩潰問題
- 語音播報
- 地震報告列表收展
- 地震時自動關閉設定頁面
- 同心圓計算公式錯誤
- 地震詳細資訊 CWA 報告頁面按鈕
- 地震報告規模顏色顯示錯誤
- 重播時接收到非台灣地震預警
- 展開地震詳細資訊時點擊首頁按鈕不會隱藏地震詳細資訊物件
- 在地圖點擊 TAB 導致異常
- 地震報告列表調整顯示為 20 筆
- TOS 服務條款更新
