# 封裝

封裝 擴充 為 `.cdps` 檔案

## 準備

資料夾看起來應該像下方這樣

```
example/
    cdps.json   // 擴充基本資訊
    main.py     // 入口點
```

## 指令

使用 `cdps pack --name example` 打包 **example** 資料夾為 **example.cdps** 檔案

:::note

`--name` 後，帶上資料夾名稱

:::

:::tip

原始程式碼下指令為 `python main.py pack --name example`

:::
