# 擴充開發

- 並不是所有想要的功能，都有人開發，如果有能力，不妨試試動手做看看！！！

## 準備

- - [x] Python ( 推薦 `>= 3.10.0` )
- - [x] VSCode ( 或其他 IDE )
- - [x] Git
- - [ ] GitHub Desktop (可選)

## 虛擬環境

- 建議使用 Python 自帶的 虛擬環境 功能，開發擴充。

:::note

- 使用 Source Code 執行 CDPS 的前提下。

:::

#### 建立 虛擬環境

```bash
python -m venv dev
```

#### 進入 虛擬環境 位置

```bash
cd ./dev/Scripts
```

#### 啟動 虛擬環境

```bash
activate.bat
```

#### 退回 根目錄

```bash
cd ../../
```

#### 安裝相關 依賴

```bash
pip install -r requirements.txt
```

#### 執行

```
python main.py
```
